# Minecraft Mod Manager
## An Mod Editor For MC Java

I found that when I installed a new mod it prevented old mods from starting. I also realize that it is difficult to constantly swap modes. I developed a simple application to solve this problem. I hope it is useful to you!

I will release 1.0 and update README.md asap.

# Features
* Customizable Pages
* Mod Editor
* Loader Installer
* Automaticly Detecting Mods
* and more! (i think)

Details are on the wiki
